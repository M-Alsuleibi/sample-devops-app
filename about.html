About This Application

This project is designed for junior DevOps trainees to practice building CI/CD pipelines. It simulates a realistic web application with a Node.js backend and a simple static frontend.

Highlights

  • Express.js REST API
  • Responsive HTML/CSS/JS frontend
  • Unit tests with Jest + Supertest
  • Dockerized with multi-stage builds
  • Security headers and basic rate limiting
  • Health and metrics endpoints for observability
